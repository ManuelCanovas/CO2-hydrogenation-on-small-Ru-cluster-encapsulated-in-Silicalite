DFT Catalyst Modeling Repository
This repository serves as a centralized collection of Density Functional Theory (DFT) output files and Microkinetic Modeling (MkM) input parameters for the CO2 hydrogenation on small clusters encapsulated in Silicalite.

Repository Structure
The data is organized into two primary directories to separate raw quantum mechanical outputs from kinetic parameter sets:

/DFT-data: Contains the raw calculation outputs, specifically OSZICAR files for electronic convergence tracking and CONTCAR files for optimized structural geometries.

/MkM_Inputs: Contains the thermodynamic and kinetic parameter sets (e.g., adsorption energies, activation barriers, and pre-exponential factors) required to execute Microkinetic Modeling simulations for the different clusters and at the different temperatrues considered.
